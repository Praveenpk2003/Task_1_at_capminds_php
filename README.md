Task Requirements

Part 1: Simple Calculator Using Functions

Declare two variables:
$a = 25;
$b = 5;

Create functions:

add(\$a, \$b)
subtract(\$a, \$b)
multiply(\$a, \$b)
divide(\$a, \$b)

Display results using echo
Expected Output

----- Calculator Results -----
Addition: 30
Subtraction: 20
Multiplication: 125
Division: 5




Part 2: Student Grade Calculator

Declare marks:
$mark1 = 78;
$mark2 = 85;
$mark3 = 90;

Calculate total and average
Apply grading logic using if/else
Average >= 90 → Grade A
Average >= 75 → Grade B
Average >= 50 → Grade C
Else → Fail

Expected Output
----- Student Grade Calculation -----
Total Marks : 253
Average : 84.33
Grade : B
